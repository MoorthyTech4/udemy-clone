A simple Udemy-inspired clone built using HTML and CSS Flexbox. This project focuses on responsive layouts, clean design, and the fundamentals of front-end development without any frameworks.

Built entirely with HTML & CSS (no JavaScript, no frameworks)

Demonstrates Flexbox mastery for alignment and spacing

Clean and minimal UI design inspired by Udemy

Beginner-friendly project to understand layout fundamentals

Can be extended with JavaScript or backend for full functionality

This project replicates a basic course platform interface for learning and practice purposes.
